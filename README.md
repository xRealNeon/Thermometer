## Features
- Mobile web app compatible
- API
- FlatUI dashboard

## Use the API
### Get the temperature: `GET` yoururl/json?k=yourkey

#### Response
| Name          | Type          | Example     |
| ------------- |:-------------:|-------------|
| in            | float         | 22.45       |
| out           | float         | 15.32       |
